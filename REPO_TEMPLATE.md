# Setup Template Repository

This repository is being used as a template for the monorepo setup with Prisma and Better Auth.

**Repository Link:** [pnpm-monorepo-prisma-better-auth-template](https://github.com/ShivaReddyVanja/pnpm-monorepo-prisma-better-auth-template.git)

## Included Components
- **Backend**: `apps/api` (Express.js)
- **Database**: `packages/database` (Prisma)

## Ignored Components (for current workspace)
- **Agent**: `apps/agent`
- **Frontend**: `apps/web` (Next.js)
- **Prisma Client**: `**/generated/` (Globally ignored)
